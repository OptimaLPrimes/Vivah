# **App Name**: Vivah Utsav

## Core Features:

- Landing Page: Display Vaishnavi & Suraj's names, photos, wedding date (December 12, 2025, 1:12 PM), and festive Indian wedding background.
- Digital Invitation Card: Showcase an interactive invitation card with animated floral borders and ethnic elements.
- Downloadable PDF: Offer the option to download the invitation card as a printable PDF.
- Venue Details with Map: Provide the venue name, full address, and an embedded Google Map for directions.
- Event Schedule: Present a timeline of wedding-related functions with dates, times, and descriptions.
- RSVP Form: Enable guests to submit their RSVP with name, email, contact, dietary preferences, and number of guests. Store the guests in Firestore.
- Guestbook AI Message Enhancement: Allow guests to leave wishes, but use an LLM tool that reasons when or if it incorporates cultural or personalized information to enhance messages, store in Firestore.

## Style Guidelines:

- Primary color: Deep saffron (#FF9933), evoking warmth, auspiciousness, and energy.
- Background color: Very pale yellow (#FAF9F0), provides a soft, elegant backdrop without overpowering the vibrant content.
- Accent color: Royal blue (#4169E1), contrasting the primary color, communicating trustworthiness.
- Headline font: 'Playfair', a modern serif lending an elegant and high-end feel.
- Body font: 'PT Sans' (sans-serif) to complement the headline font.
- Use ornate icons with Indian wedding motifs, like kalash, diyas, paisleys, and traditional sweets.
- Design a mobile-friendly layout using a responsive framework with clear content hierarchy.
- Subtle animations like floral border animations, image transitions, and scroll effects.
